Fuzzy C-Mean Clustering#

Fuzzy C-Means adalah salah satu teknik clustering yang berdasarkan prinsip fuzzy logic, di mana setiap data tidak hanya menjadi bagian dari satu kelompok secara tegas, tetapi bisa menjadi anggota dari beberapa kluster sekaligus dengan tingkat keanggotaan tertentu. Ini membedakannya dari metode seperti K-Means yang hanya membolehkan keanggotaan tunggal.#

Konsep Dasar Fuzzy C-Means Clustering#

Fuzzy C-Means (FCM) adalah salah satu algoritma clustering atau pengelompokan data yang menggunakan pendekatan logika fuzzy, di mana satu data dapat termasuk dalam lebih dari satu kluster pada saat yang sama, dengan tingkat keanggotaan tertentu (nilai antara 0 dan 1).

Beberapa komponen penting dalam algoritma FCM meliputi:

  • Titik Data (Data Point): Sekumpulan objek yang akan diklasifikasikan ke dalam beberapa kelompok.

  • Titik Pusat (Centroid): Titik yang berfungsi sebagai pusat dari masing-masing kelompok dalam proses pengelompokan.

  • Matriks Keanggotaan Fuzzy (U): Matriks yang memuat derajat keanggotaan setiap titik data terhadap masing-masing kelompok.

  • Parameter Fuzziness (m): Parameter yang menentukan tingkat ketidakpastian dalam keanggotaan suatu data terhadap kelompok tertentu. Umumnya bernilai 2; semakin besar nilai m, maka derajat keanggotaan data terhadap suatu kelompok menjadi semakin tidak tegas.

Perbedaan dengan K-Means

Pada metode klasterisasi seperti K-Means, setiap data hanya bisa menjadi anggota satu kluster secara pasti (hard clustering). Sedangkan dalam FCM, keanggotaan data terhadap kluster bersifat relatif (soft clustering) — artinya satu data bisa “tertarik” ke beberapa kluster, tetapi dengan bobot atau derajat keanggotaan yang berbeda-beda.

Algoritma Fuzzy C-Means Clustering#

Fuzzy C-Means adalah algoritma soft clustering yang bekerja dengan cara memperbarui pusat kluster dan nilai keanggotaan fuzzy secara berulang, hingga hasilnya stabil. Setiap data dapat menjadi anggota lebih dari satu kluster, dengan nilai keanggotaan yang menunjukkan kedekatannya ke masing-masing pusat kluster.

Langkah-langkah Algoritma FCM#

1. Inisialisasi#

  • Tentukan jumlah kluster \(c\)

  • Pilih nilai parameter fuzziness \(m\) (biasanya \(m = 2\))

  • Buat matriks keanggotaan awal \(U_{ij}\) secara acak, dengan syarat:

    \[ \sum_{j=1}^c U_{ij} = 1, \quad \forall i \]

2. Hitung Pusat Kluster#

\[ v_j = \frac{\sum_{i=1}^n (U_{ij})^m \cdot x_i}{\sum_{i=1}^n (U_{ij})^m} \]

Di mana:

  • \(v_j\) adalah pusat cluster ke-j

  • \(x_i\) adalah data ke-i

  • \(U_{ij}\) adalah nilai keanggotaan data ke-i terhadap cluster ke-j

3. Update derajat keanggotaan#

Untuk setiap data \(x_i\) dan kluster \(j\), perbarui derajat keanggotaan:

\[ u_{ij} = \frac{1}{\sum_{k=1}^{c} \left( \frac{\|x_i - c_j\|}{\|x_i - c_k\|} \right)^{\frac{2}{m - 1}}} \]

Di mana:

* \(|x_i - v_j|\) adalah jarak antara data ke-i dengan pusat cluster ke-j

4. Cek Konvergensi#

  • Hitung perubahan antara matriks keanggotaan sebelumnya dan yang baru.

  • Jika selisihnya lebih kecil dari nilai toleransi \(\epsilon\), hentikan iterasi.

  • Jika belum, ulangi langkah 2 dan 3.

5. Penentuan Hasil#

  • Setiap data dikaitkan dengan kluster yang memiliki nilai keanggotaan tertinggi.

  • Hasil akhirnya adalah:

    • Pusat kluster akhir

    • Matriks keanggotaan akhir

    • Label kluster dominan untuk tiap data (jika diperlukan)

Data dan Parameter#

  • Jumlah cluster: \(c = 2\)

  • Tingkat fuzziness: \(m = 2\)

Langkah 1: Inisialisasi Matriks Keanggotaan \(U^{(0)}\)#

Ukuran matriks: \(2 \times 4\)

Misalkan:

\[\begin{split} U^{(0)} = \begin{bmatrix} 0.6 & 0.4 & 0.9 & 0.1 \\ 0.4 & 0.6 & 0.1 & 0.9 \end{bmatrix} \end{split}\]

Catatan: Setiap kolom jumlahnya 1, misal \(0.6 + 0.4 = 1\).

Langkah 2: Hitung Pusat Cluster (Centroid) \(v_j\)#

Rumus centroid cluster ke-\(j\):

\[ v_j = \frac{\sum_{i=1}^n (u_{ji})^m \cdot x_i}{\sum_{i=1}^n (u_{ji})^m} \]

Hitung \(v_1\):#

\[\begin{split} \begin{aligned} v_1 &= \frac{(0.6)^2 \times 1 + (0.4)^2 \times 2 + (0.9)^2 \times 5 + (0.1)^2 \times 6}{(0.6)^2 + (0.4)^2 + (0.9)^2 + (0.1)^2} \\ &= \frac{0.36 \times 1 + 0.16 \times 2 + 0.81 \times 5 + 0.01 \times 6}{0.36 + 0.16 + 0.81 + 0.01} \\ &= \frac{0.36 + 0.32 + 4.05 + 0.06}{1.34} \\ &= \frac{4.79}{1.34} \approx 3.574 \end{aligned} \end{split}\]

Hitung \(v_2\):#

\[\begin{split} \begin{aligned} v_2 &= \frac{(0.4)^2 \times 1 + (0.6)^2 \times 2 + (0.1)^2 \times 5 + (0.9)^2 \times 6}{(0.4)^2 + (0.6)^2 + (0.1)^2 + (0.9)^2} \\ &= \frac{0.16 \times 1 + 0.36 \times 2 + 0.01 \times 5 + 0.81 \times 6}{0.16 + 0.36 + 0.01 + 0.81} \\ &= \frac{0.16 + 0.72 + 0.05 + 4.86}{1.34} \\ &= \frac{5.79}{1.34} \approx 4.321 \end{aligned} \end{split}\]

Langkah 3: Update Matriks Keanggotaan \(U\)#

Rumus update \(u_{ji}\):

\[ u_{ji} = \frac{1}{\sum_{k=1}^c \left( \frac{d(x_i, v_j)}{d(x_i, v_k)} \right)^{\frac{2}{m-1}}} \]

dengan \(d(x_i, v_j) = |x_i - v_j|\)

Contoh hitung untuk data \(x_1 = 1\):#

  • \(d(x_1, v_1) = |1 - 3.574| = 2.574\)

  • \(d(x_1, v_2) = |1 - 4.321| = 3.321\)

Hitung \(u_{1,1}\):

\[ u_{1,1} = \frac{1}{\left(\frac{2.574}{2.574}\right)^2 + \left(\frac{2.574}{3.321}\right)^2} = \frac{1}{1 + (0.775)^2} = \frac{1}{1 + 0.6006} = \frac{1}{1.6006} = 0.625 \]

Maka \(u_{2,1} = 1 - 0.625 = 0.375\).

Hitung untuk \(x_2 = 2\):#

  • \(d(x_2, v_1) = |2 - 3.574| = 1.574\)

  • \(d(x_2, v_2) = |2 - 4.321| = 2.321\)

\[ u_{1,2} = \frac{1}{1 + \left(\frac{1.574}{2.321}\right)^2} = \frac{1}{1 + (0.678)^2} = \frac{1}{1 + 0.460} = \frac{1}{1.460} = 0.685 \]

Maka \(u_{2,2} = 0.315\).

Hitung untuk \(x_3 = 5\):#

  • \(d(x_3, v_1) = |5 - 3.574| = 1.426\)

  • \(d(x_3, v_2) = |5 - 4.321| = 0.679\)

\[ u_{1,3} = \frac{1}{1 + \left(\frac{1.426}{0.679}\right)^2} = \frac{1}{1 + (2.101)^2} = \frac{1}{1 + 4.414} = \frac{1}{5.414} = 0.185 \]

Maka \(u_{2,3} = 0.815\).

Hitung untuk \(x_4 = 6\):#

  • \(d(x_4, v_1) = |6 - 3.574| = 2.426\)

  • \(d(x_4, v_2) = |6 - 4.321| = 1.679\)

\[ u_{1,4} = \frac{1}{1 + \left(\frac{2.426}{1.679}\right)^2} = \frac{1}{1 + (1.445)^2} = \frac{1}{1 + 2.088} = \frac{1}{3.088} = 0.324 \]

Maka \(u_{2,4} = 0.676\).

Matriks keanggotaan baru \(U^{(1)}\):#

\[\begin{split} U^{(1)} = \begin{bmatrix} 0.625 & 0.685 & 0.185 & 0.324 \\ 0.375 & 0.315 & 0.815 & 0.676 \end{bmatrix} \end{split}\]

Langkah 4: Cek Konvergensi#

Hitung centroid baru \(v_j^{(1)}\) dan bandingkan dengan \(v_j^{(0)}\). Jika perubahan < 0.01, berhenti. Jika tidak, ulangi.

Hitung Centroid Baru \(v_1^{(1)}\):#

\[\begin{split} \begin{aligned} v_1^{(1)} &= \frac{0.625^2 \times 1 + 0.685^2 \times 2 + 0.185^2 \times 5 + 0.324^2 \times 6}{0.625^2 + 0.685^2 + 0.185^2 + 0.324^2} \\ &= \frac{0.3906 \times 1 + 0.4692 \times 2 + 0.0342 \times 5 + 0.1049 \times 6}{0.3906 + 0.4692 + 0.0342 + 0.1049} \\ &= \frac{0.3906 + 0.9384 + 0.171 + 0.6294}{0.999} \\ &= \frac{2.1294}{0.999} \approx 2.132 \end{aligned} \end{split}\]

Hitung Centroid Baru \(v_2^{(1)}\):#

\[\begin{split} \begin{aligned} v_2^{(1)} &= \frac{0.375^2 \times 1 + 0.315^2 \times 2 + 0.815^2 \times 5 + 0.676^2 \times 6}{0.375^2 + 0.315^2 + 0.815^2 + 0.676^2} \\ &= \frac{0.1406 \times 1 + 0.0992 \times 2 + 0.6642 \times 5 + 0.4569 \times 6}{0.1406 + 0.0992 + 0.6642 + 0.4569} \\ &= \frac{0.1406 + 0.1984 + 3.321 + 2.7414}{1.3609} \\ &= \frac{6.4014}{1.3609} \approx 4.703 \end{aligned} \end{split}\]

Langkah 5: Hitung Fungsi Objektif \(J_m\)#

\[ J_m = \sum_{j=1}^c \sum_{i=1}^n (u_{ji})^m \cdot d^2(x_i, v_j) \]

Hitung \(J_m\) pada iterasi 1:

\[\begin{split} \begin{aligned} J_m &= \sum_{i=1}^4 (u_{1i})^2 \cdot (x_i - v_1)^2 + \sum_{i=1}^4 (u_{2i})^2 \cdot (x_i - v_2)^2 \\ &= \text{untuk } j=1: \\ &0.625^2 \times (1 - 3.574)^2 + 0.685^2 \times (2 - 3.574)^2 + 0.185^2 \times (5 - 3.574)^2 + 0.324^2 \times (6 - 3.574)^2 \\ &= 0.3906 \times 6.625 + 0.4692 \times 2.478 + 0.0342 \times 2.037 + 0.1049 \times 5.898 \\ &= 2.588 + 1.163 + 0.070 + 0.618 \\ &= 4.439 \\ &\text{untuk } j=2: \\ &0.375^2 \times (1 - 4.321)^2 + 0.315^2 \times (2 - 4.321)^2 + 0.815^2 \times (5 - 4.321)^2 + 0.676^2 \times (6 - 4.321)^2 \\ &= 0.1406 \times 11.05 + 0.0992 \times 5.39 + 0.6642 \times 0.46 + 0.4569 \times 2.83 \\ &= 1.554 + 0.535 + 0.305 + 1.292 \\ &= 3.686 \\ &J_m = 4.439 + 3.686 = 8.125 \end{aligned} \end{split}\]

Implementasi Python#

import numpy as np
import pandas as pd

# Data dan parameter
X = np.array([1, 2, 5, 6])  # Data satu dimensi
c = 2                       # Jumlah cluster
m = 2.0                     # Tingkat fuzziness
epsilon = 0.01              # Ambang batas konvergensi
max_iter = 100              # Maksimum iterasi

# Inisialisasi keanggotaan U secara manual (hasil perhitungan manual awal)
U = np.array([
    [0.6, 0.4, 0.9, 0.1],
    [0.4, 0.6, 0.1, 0.9]
])

# Fungsi untuk menghitung centroid
def calculate_centroids(U, X, m):
    um = U ** m
    return (um @ X) / um.sum(axis=1)

# Fungsi untuk memperbarui matriks keanggotaan U
def update_membership(X, V, m):
    n = len(X)
    c = len(V)
    U_new = np.zeros((c, n))
    for i in range(n):
        for j in range(c):
            denom_sum = 0.0
            for k in range(c):
                dist_ratio = abs(X[i] - V[j]) / abs(X[i] - V[k]) if abs(X[i] - V[k]) != 0 else 1e-10
                denom_sum += dist_ratio ** (2 / (m - 1))
            U_new[j, i] = 1.0 / denom_sum
    return U_new

# Fungsi untuk menghitung fungsi objektif
def objective_function(U, V, X, m):
    obj = 0.0
    for j in range(c):
        for i in range(len(X)):
            obj += (U[j, i] ** m) * ((X[i] - V[j]) ** 2)
    return obj

# Proses iterasi
history = []
for iteration in range(max_iter):
    V = calculate_centroids(U, X, m)
    obj = objective_function(U, V, X, m)
    history.append((iteration + 1, V.copy(), U.copy(), obj))

    U_new = update_membership(X, V, m)
    V_new = calculate_centroids(U_new, X, m)

    if np.max(np.abs(V - V_new)) < epsilon:
        break
    U = U_new

# Hasil ke DataFrame agar mudah dibaca
results = []
for iter_num, centroids, membership, obj_val in history:
    result = {
        "Iterasi": iter_num,
        "Centroid 1": centroids[0],
        "Centroid 2": centroids[1],
        "Objektif": obj_val
    }
    for j in range(c):
        for i in range(len(X)):
            result[f"U[{j+1},{i+1}]"] = membership[j, i]
    results.append(result)

df_results = pd.DataFrame(results)
print(df_results.round(4))  # Pembulatan untuk keterbacaan
